Responsive Redesign

Usability Issues
  • Efficent to use! 5 menu buttons near the top of the page that allow the user to navigate to different pages
  • Could be made more minimalistic by condensing menu buttons into a hamburger menu
  • Not as learnable as it could be. Given that there are only a few actions that a user can take, such as donate or connect, it should be more learnable
  • The tiles at the bottom of the page do not look like buttons -> less learnable
  • The website is not very memorable. The front page feels simultaneously cluttered and blank. There is also not much color in the whole page
The website in question is that of S.A.F.E DC, a nonprofit tennis organization I used to volunteer for.
See the original site here!
Accessibility Tests
  • WAIVE identified that the black text on white backgroundwas a "contrast error". This did not seem important, the text is easily readable
  • It also identified text that was "too small" - but it was just copyright text
  • In summary, I had no major findings from accessibility testing
